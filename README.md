# Sigma Lang 🗿

**For the real sigmas.**

> The programming language that has absolutely no business existing... yet somehow does.

Sigma Lang is an interpreted programming language inspired by my boredom, except almost every keyword has been aggressively translated into internet brainrot.

Why write `print "Hello, world!"` when you can write `yap "Hello, world!"`. Programming should be fun, you get what I am saying, right?

## ✨ Features (better not to talk about them)

I'd tell you about the features...

...but then I'd have to fix them.

**Here are a few:**

🧠 Brainrot-inspired syntax

⚡ Tree-walk interpreter written in TypeScript

🖥️ In-browser playground

🎨 Syntax highlighting

💡 Autocomplete

👴 Classes, Inheritance and some other stuff I am trying to remember 🤔.

## Why you ask? 🤷‍♂️

Because every programming language tries to sound serious.

Sigma Lang asks the important questions.

- Why is it called `print` instead of `yap`?
- Why is it a `function` instead of a `funk`?
- Why make a `class` when you can start a `gang`?
